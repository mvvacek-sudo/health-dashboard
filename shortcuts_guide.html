🍎 Apple Shortcuts — Health Sync

Automatický import dat z Apple Health do dashboardu každý den ráno.

1Otevři Apple Shortcuts
Aplikace Zkratky (Shortcuts) → tlačítko + vpravo nahoře → nová zkratka.
2Přidej akci: Najít vzorky zdraví
Hledat akci: Najít vzorky zdravotních dat

Přidej 4× tuto akci pro každou metriku:

Tělesná hmotnost Procento tělesného tuku Klidový srdeční tep Krevní tlak (systolický)

Pro každou nastav: Limit: 1 · Řadit: Nejnovější
3Přidej akci: Získat podrobnosti
Pro každou metriku přidej akci Získat podrobnosti vzorků zdravotních dat
Vyber: Hodnota
4Přidej akci: Načíst obsah URL
Hledat: Načíst obsah URL

URL:
https://hrneqybterqinixsyzhz.supabase.co/rest/v1/body_measurements

Metoda: POST
Hlavičky:
 • apikey → eyJhbGci... (tvůj anon key)
 • Authorization → Bearer eyJhbGci...
 • Content-Type → application/json
 • Prefer → return=representation

Tělo požadavku: JSON
{ "weight_kg": [Hodnota z Tělesná hmotnost], "body_fat_pct": [Hodnota z Procento tělesného tuku], "resting_hr": [Hodnota z Klidový srdeční tep] }
⚠️ Hodnoty z Apple Health vlož jako proměnné (Magic Variables) — klikni na místo v JSON a vyber proměnnou ze seznamu.
5Nastav automatické spouštění
Záložka Automatizace → + → Čas dne
Nastav: každý den v 8:00
Vyber svou zkratku → Spustit okamžitě (bez potvrzení)
6Hotovo ✓
Každý den ráno v 8:00 se automaticky načtou data z Apple Health a uloží do Supabase → dashboard se aktualizuje.
